Functionality


Boards

Long press the board header to open the drawer.

There you will be able to Add board, Edit board name, Delete board.

To change boards you can either press the arrow buttons or swipe to the left or to the right on the board header.


Tasks

Add a new task by pressing the + button.

You need to add a name and optionally you can add a description, select a due date and priority for the task.

You can mark a task as DONE by swiping to the right on one of the tasks.

Swipe to the left in order to delete a task.

Both actions, mark as done and delete can be canceled within 10 seconds, by pressing the undo action on the toast that appears.
